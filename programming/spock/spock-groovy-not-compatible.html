The Spock compiler plugin cannot execute because Spock 1.0.0-groovy-2.4 is not compatible with Groovy 2.3.10.

Spock framework는 Groovy를 이용하여 Unit Test를 쉽고 가독성이 좋게 작성할 수 있는 framework다.
하지만 Spock은 지원하는 Groovy Version이 명확해서 Spock을 버전업하면 그에 맞는 Groovy 버전을 같이 올려야 한다.
행여나 깜박하고 버전이 다른 경우 다음과 같이 까칠한 예외를 출력한다.

Gradle을 사용한다면 아래와 같이 최상위 build.gradle에 아래와 같이 추가한다.


    dependencies {
    compile 'org.codehaus.groovy:groovy-all:2.4.1'
    testCompile 'org.spockframework:spock-core:1.0-groovy-2.4'
    }

설정에 추가한 뒤에 IntelliJ에서는 실행이 여전히 안된다. 그래서 command shell 상에서 한번 수동으로 실행해줘야 한다.
